# sao-substruct

> A Node.js framework

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao substruct my-project
```

### From git

```bash
sao internalfx/sao-substruct my-project
```
